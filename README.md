# Cosmo HUD

Simple status HUD for FiveM and Now made for ND Core.
It now includes hunger, thirst, alcohol and stamina 

## Installation

Put 'cosmo_hud' into your 'resources' folder.

Edit config.lua to your liking.

Do not change the name or it will not work.

## Requirements

ND_Core
ND_Status
pma-voice
rp-radio (optional)

https://loading.io/progress/ for the loading-Bar library.
